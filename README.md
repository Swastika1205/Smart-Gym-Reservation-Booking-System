FitSlot – Smart Gym Reservation System

FitSlot is a full-stack gym reservation system developed using Next.js, Node.js, Express.js, TypeScript, and MySQL. The platform allows users to book gym slots in real time, check slot availability, and manage reservations efficiently.

Features

• Real-time slot booking system
• Dynamic slot availability tracking
• User booking management
• RESTful API integration
• Server-side validation and error handling
• MySQL database management
• Responsive frontend with Next.js

Tech Stack

Frontend:

• Next.js
• React.js
• TypeScript

Backend:

• Node.js
• Express.js
• TypeScript

Database:

• MySQL

Project Highlights

• Developed 8+ RESTful API endpoints for booking and user management
• Built dynamic booking logic with optimized MySQL queries
• Connected frontend components with backend APIs for seamless data flow
• Implemented validation and error handling to improve system reliability
